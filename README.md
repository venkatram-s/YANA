# YANA

Yet Another News Aggregator. It fetches news. That's it. 

It's a vibe coded app to be honest. So, there'll be bugs, of course. 

## Stack
- React 19 + Vite 7
- Node.js 24.
- IndexedDB for local storage. 

## Setup

```bash
git clone https://github.com/venkatram-s/YANA.git
cd YANA
npm install --legacy-peer-deps
npm run dev
```

Android? Just run it.
```bash
npx cap open android
```

## Features
- Grabs RSS feeds.
- AI summaries if you care.
- Vault is encrypted. AES-GCM. 
- Forget the password and it wipes everything. Don't forget the password.

It is what it is.
